## Formula Properties Reference

### Invoice Formulas

| Property | Formula | Purpose |
|----------|---------|---------|
| **Subtotal** | `prop("Hours") * prop("Rate")` | Calculates base cost |
| **Tax Amount** | `prop("Subtotal") * prop("Tax Rate")` | Calculates tax |
| **Total** | `prop("Subtotal") + prop("Tax Amount")` | Final amount due |

#### Formula Breakdown:

```
Subtotal:
  prop("Hours")     â†’ References Hours property
  *                   â†’ Multiplication
  prop("Rate")      â†’ References Rate property

Example: Hours: 10 Ã— Rate: $150 = Subtotal: $1,500
```

```
Tax Amount:
  prop("Subtotal")       â†’ Uses calculated subtotal
  * 
  prop("Tax Rate")       â†’ References tax rate (e.g., 0.08)

Example: $1,500 Ã— 0.08 = $120
```

```
Total:
  prop("Subtotal") + prop("Tax Amount")
  
Example: $1,500 + $120 = $1,620
```
